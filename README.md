Titanic Dataset - Exploratory Data Analysis (EDA)

📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset using Python and Pandas.

The main objective of this project is to explore the dataset, identify missing values, analyze passenger information, and extract meaningful insights.

🛠️ Tools and Libraries Used

- Python
- Pandas

📊 Analysis Performed

- Dataset Overview ("head()")
- Dataset Shape ("shape")
- Column Names ("columns")
- Dataset Information ("info()")
- Statistical Summary ("describe()")
- Missing Values Analysis ("isnull().sum()")
- Gender Count Analysis
- Survival Count Analysis
- Passenger Class Analysis
- Average, Maximum, and Minimum Age Analysis
- Average Fare Analysis

🔍 Key Findings

- The dataset contains passenger information from the Titanic disaster.
- Some columns contain missing values.
- Most passengers belonged to the third class.
- The dataset provides insights into passenger demographics and survival patterns.
