📝 Project Overview

This project focuses on Exploratory Data Analysis (EDA) of the Titanic dataset as part of the CodeAlpha Data Analytics Internship.

The main objective of this project is to explore and understand the dataset, identify missing values, analyze passenger information, and extract meaningful insights using Python and Pandas.

EDA helps in understanding the structure of data before performing advanced analysis or machine learning.

✨ Key Features

- Dataset Overview using "head()"
- Dataset Shape Analysis
- Column Names Identification
- Missing Values Detection
- Statistical Summary Generation
- Gender Distribution Analysis
- Survival Count Analysis
- Passenger Class Analysis
- Average, Maximum, and Minimum Age Analysis
- Average Fare Analysis
- Unique Values Analysis

🛠️ Technologies Used

- Python
- Pandas
- VS Code

📊 Analysis Performed

✔️ Dataset Overview ("head()")

✔️ Dataset Information ("info()")

✔️ Statistical Summary ("describe()")

✔️ Missing Values Analysis ("isnull().sum()")

✔️ Gender Count Analysis

✔️ Survival Count Analysis

✔️ Passenger Class Analysis

✔️ Age and Fare Analysis

🔍 Key Findings

- The dataset contains passenger information from the Titanic disaster.
- Some columns contain missing values, especially the Cabin column.
- Most passengers belonged to the third class.
- The dataset provides insights into passenger demographics and survival patterns.
